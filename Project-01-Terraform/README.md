# Terraform Azure Infrastructure Deployment

## Project Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform to deploy Azure infrastructure.

## Resources Created

- Azure Resource Group
- Azure Virtual Network
- Azure Subnet
- Azure Storage Account

## Project Structure

```
Project-01-Terraform/
│
├── provider.tf
├── variables.tf
├── main.tf
├── outputs.tf
└── README.md
```

## Terraform Commands

```bash
terraform init
terraform fmt
terraform validate
terraform plan
terraform apply
terraform destroy
```
