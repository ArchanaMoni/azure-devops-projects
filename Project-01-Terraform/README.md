# Terraform Azure Infrastructure Project

## Project Overview

This project provisions Azure infrastructure using Terraform.

## Resources Created

- Azure Resource Group
- Azure Virtual Network
- Azure Subnet
- Azure Storage Account

## Technologies Used

- Terraform
- Microsoft Azure
- AzureRM Provider

## Terraform Commands

```bash
terraform init
terraform validate
terraform plan
terraform apply
terraform destroy
```

## Project Structure

- provider.tf
- variables.tf
- main.tf
- outputs.tf

## Learning Outcome

- Learned Infrastructure as Code (IaC)
- Created Azure resources using Terraform
- Used variables and outputs
- Managed Azure infrastructure with Terraform
