# Terraform Interview Questions

## 1. What is Terraform?

Terraform is an Infrastructure as Code (IaC) tool used to provision and manage infrastructure using code.

---

## 2. Why do we use Terraform?

- Infrastructure as Code
- Automation
- Reusability
- Version Control
- Consistency

---

## 3. What is provider.tf?

Defines the cloud provider Terraform will use.

Example:
- AzureRM
- AWS
- Google Cloud

---

## 4. What is variables.tf?

Defines reusable input variables.

---

## 5. What is main.tf?

Contains the infrastructure resources to be created.

---

## 6. What is outputs.tf?

Displays useful information after deployment.

Example:
- Resource Group Name
- Storage Account Name
- Resource ID

---

## 7. Terraform Commands

terraform init

terraform validate

terraform plan

terraform apply

terraform destroy
