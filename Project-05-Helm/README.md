# Helm Deployment Project

## Project Overview

This project demonstrates deploying a Kubernetes application using Helm.

## Technologies Used

- Helm
- Kubernetes
- Docker Desktop
- kubectl
- Git
- GitHub

## Features Implemented

- Helm Installation
- Helm Chart Creation
- Chart.yaml
- values.yaml
- Templates
- Helm Template Rendering
- Helm Install
- Helm Upgrade
- Helm Rollback
- Helm History

## Helm Commands Used

```bash
helm create my-nginx-chart

helm template my-release my-nginx-chart

helm install my-release my-nginx-chart

helm list

helm history my-release

helm upgrade my-release my-nginx-chart

helm rollback my-release 1
```
