# Helm Interview Questions

## 1. What is Helm?

Helm is the package manager for Kubernetes that simplifies application deployment using reusable charts.

---

## 2. What is a Helm Chart?

A Helm Chart is a package containing Kubernetes templates and configuration files.

---

## 3. What is Chart.yaml?

Chart.yaml stores metadata such as chart name, version, and application version.

---

## 4. What is values.yaml?

values.yaml contains configurable values used by Helm templates.

---

## 5. What is a Helm Release?

A Helm Release is an installed instance of a Helm Chart.

---

## 6. Difference between Chart and Release?

Chart is a blueprint.

Release is a deployed instance.

---

## 7. What is helm upgrade?

Updates an existing Helm Release.

---

## 8. What is helm rollback?

Restores a previous release revision.

---

## 9. What is helm history?

Displays the revision history of a Helm Release.

---

## 10. Difference between helm template and helm install?

helm template generates Kubernetes YAML without deploying.

helm install deploys the application to Kubernetes.
