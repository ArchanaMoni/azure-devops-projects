# 📊 Project-06: Prometheus & Grafana Monitoring

## 📌 Project Overview

This project demonstrates infrastructure monitoring using Prometheus, Grafana, and Node Exporter.

Prometheus collects metrics from the local machine through Node Exporter, while Grafana visualizes these metrics using dashboards.
