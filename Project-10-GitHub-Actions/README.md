# 🚀 Project-10: GitHub Actions

## Project Overview

This project demonstrates a CI pipeline using GitHub Actions.

The workflow automatically:

- Checks out the repository
- Displays runner information
- Builds a Docker image
- Verifies the Docker image
- Completes the pipeline successfully

## Technologies Used

- GitHub Actions
- Docker
- Git
- GitHub

## Workflow

Git Push
   ↓
GitHub Actions
   ↓
Checkout Repository
   ↓
Build Docker Image
   ↓
Verify Docker Image
   ↓
Pipeline Completed
